## 📋 كيفية الاستخدام في FlutLab

1. ارفع ملف ZIP
2. انتظر تحميل المشروع
3. اضغط **Run** مباشرة

## 🚀 كيفية البناء محلياً

```bash
# 1. تثبيت المتطلبات
flutter pub get

# 2. تشغيل التطبيق
flutter run

# 3. بناء APK
flutter build apk --release
```
